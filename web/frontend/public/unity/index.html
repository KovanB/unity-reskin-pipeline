Unity WebGL build goes here. Once built, your game will be playable right in this frame.
  1. Copy scripts from templates/unity-scripts/ into your Unity project
  2. Open Unity → File → Build Settings → WebGL → Build
  3. Copy the Build/ folder to web/frontend/public/unity/
  4. Redeploy — the game plays live with customization!
